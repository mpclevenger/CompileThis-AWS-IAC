LambdaTasks API
A lightweight, serverless task management API built using AWS Lambda, API Gateway, and DynamoDB. It supports full CRUD operations and is designed with scalable, event-driven architecture in mind. Monitoring and logging are handled with AWS CloudWatch, and IAM roles are used to secure access between services.

ShortStack
A cloud-native URL shortener that transforms long links into compact, shareable URLs, powered by AWS Lambda and DynamoDB. It features real-time redirection via API Gateway and tracks usage metrics, stored in DynamoDB and visualized through CloudWatch dashboards. The project demonstrates serverless application design and data modeling for high availability.
